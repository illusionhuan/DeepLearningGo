# DeepLearningGo基于深度强化学习的围棋AI
## 使用说明

下载python3.10版本并添加到环境变量中

在当前路径下打开终端并执行以下命令:

```shell
# 更新pip版本
python.exe -m pip install --upgrade pip
# 安装项目所需依赖包
python.exe -m pip install -r ./requirements.txt
```

下载完成后进入dlGame文件夹下的httpfrontend目录下的run_web.py文件，右键点击运行run_web就可以启动项目,项目部署成功后，就可以使用网页来进行围棋对弈

网页地址：http://127.0.0.1:5000
